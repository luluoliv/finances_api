Welcome to the Finances Manager API Documentation

This is the documentation for the Finances Manager API.

Base URL

The base URL of the API is https://finances-manager.onrender.com/. All the endpoints listed below should be appended to this base URL.

Endpoints

  • /incomes/
  • /expenses/
  • /goals/

Endpoints for Incomes

List all incomes and create a new one. Endpoint: /incomes/ (GET/POST)

Retrieve, update, and delete a specific income. Endpoint: /incomes/{id}/ (GET/PUT/DELETE)

Endpoints for Expenses

List all expenses and create a new one. Endpoint: /expenses/ (GET/POST)

Retrieve, update, and delete a specific expense. Endpoint: /expenses/{id}/ (GET/PUT/DELETE)

Endpoints for Goals

List all goals and create a new one. Endpoint: /goals/ (GET/POST)

Retrieve, update, and delete a specific goal. Endpoint: /goals/{id}/ (GET/PUT/DELETE)
